# NumPy Learning 📊

This repository contains my practice and learning files for NumPy.

## About NumPy
NumPy (Numerical Python) is a powerful Python library used for numerical computing. 
It provides support for arrays, matrices, and many mathematical operations.

## Topics I Learned
- Creating NumPy arrays
- Array indexing and slicing
- Mathematical operations
- Array reshaping
- Random number generation
- Statistical functions

## Files in this Repository
- day1_arrays.py – Creating and understanding arrays
- day2_operations.py – Mathematical operations on arrays
- day3_indexing.py – Indexing and slicing arrays
- day4_random.py – Random numbers in NumPy
- day5_statistics.py – Mean, median, standard deviation
